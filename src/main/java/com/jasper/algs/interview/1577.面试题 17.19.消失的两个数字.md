## 17.19. 消失的两个数字

给定一个数组，包含从 1 到 N 所有的整数，但其中缺了两个数字。你能在 O(N) 时间内只用 O(1) 的空间找到它们吗？

以任意顺序返回这两个数字均可。

示例 1:

```
输入: [1]
输出: [2,3]
```

示例 2:

```
输入: [2,3]
输出: [1,4]
```

提示：

```
    nums.length <= 30000
```

##### 一：异或+位掩码（推荐）

基本思路：

**异或，消除出现偶数次的数据，保留出现奇数次的数据**

``` java
0 ^ a = a
a ^ a = 0
```

**取负相与，保留位中最右边 1 ，且将其余的 1 设位 0 **
