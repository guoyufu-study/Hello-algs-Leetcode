## 0004.寻找两个有序数组的中位数

### 问题描述

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。

请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 \\(\mathrm{O}(log(m + n))\\)。

你可以假设 nums1 和 nums2 **不会同时为空**。

示例 1:

```
nums1 = [1, 3]
nums2 = [2]

则中位数是 2.0
```

示例 2:

```
nums1 = [1, 2]
nums2 = [3, 4]

则中位数是 (2 + 3)/2 = 2.5
```

中位数（Median）又称中值，是按顺序排列的一组数据中居于中间位置的数。它可以将数值集合划分为相等的上下两部分。如果观察值有偶数个，通常取最中间的两个数值的平均数作为中位数。
中位数 = 中间位置上的数。
