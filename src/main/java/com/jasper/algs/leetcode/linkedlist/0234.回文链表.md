### 问题描述

请判断一个链表是否为 **回文链表**。

示例 1:

```
输入: 1->2
输出: false
```

示例 2:

```
输入: 1->2->2->1
输出: true
```

进阶：
* 你能否用 \\(\mathrm{O}(n)\\) 时间复杂度和 \\(\mathrm{O}(1)\\) 空间复杂度解决此题？
