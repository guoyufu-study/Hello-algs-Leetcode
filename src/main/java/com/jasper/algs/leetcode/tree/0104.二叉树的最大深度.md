## 0104. 二叉树的最大深度

[刷题](qu0104/solu/Solution.java)

与 [剑指 Offer 55 - I. 二叉树的深度](../../sword2offer/剑指 Offer 55 - I.二叉树的深度.md) 完全相同

### 问题描述

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

示例：

```
给定二叉树 [3,9,20,null,null,15,7]，

    3
   / \
  9  20
    /  \
   15   7

返回它的最大深度 3 。
```
