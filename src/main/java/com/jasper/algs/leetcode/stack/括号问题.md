### 术语

**有效括号字符串**：空 ""、"(" + A + ")" 或 A + B，其中 A 和 B 都是有效的括号字符串，+ 代表字符串的连接。例如，""，"()"，"(())()" 和 "(()(()))" 都是有效的括号字符串。

如果有效字符串 S 非空，且不存在将其拆分为 S = A+B 的方法，我们称其为 **原语（primitive）**，其中 A 和 B 都是非空有效括号字符串。

将一个非空有效字符串 S 进行 **原语化分解**，使得：\\(S = P_1 + P_2 + ... + P_k\\)，其中 \\(P_i\\) 是有效括号字符串原语。
