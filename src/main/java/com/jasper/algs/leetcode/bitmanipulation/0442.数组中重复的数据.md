## 0442. 数组中重复的数据

### 问题描述

给定一个整数数组 a，其中`1 ≤ a[i] ≤ n` （n为数组长度）, 其中有些元素出现**两次**而其他元素出现**一次**。

找到所有出现两次的元素。

你可以**不用到任何额外空间**并在\\(\mathrm{O}(n)\\)时间复杂度内解决这个问题吗？

示例：

```
输入:
[4,3,2,7,8,2,3,1]

输出:
[2,3]
```
