#### 0011.盛最多水的容器

1. 暴力求解

2. 双指针（推荐）

  盛多少水取决于两条线构成的矩形的面积；
  
  矩形面积 = 长度 * 宽度；较短线段的长度将会作为矩形区域的宽度，两线间距将会作为矩形区域的长度；
  
  先计算长度最大时，矩形面积；然后，逐渐缩小长度，增大宽度，来找出面积更大的矩形。
