### 状态转移公式

#### 组合问题公式

$$dp[i] += dp[i-num]$$

#### True、False问题公式

$$dp[i] = dp[i] or dp[i-num]$$

#### 最大最小问题公式

$$dp[i] = min(dp[i], dp[i-num]+1)$$

$$dp[i] = max(dp[i], dp[i-num]+1)$$
