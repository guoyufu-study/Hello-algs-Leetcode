### 问题描述

给你一个整数数组 `A`，请你给数组中的每个元素 `A[i]` 都加上一个任意数字 `x` \\(（-K \le x \le K）\\)，从而得到一个新数组 `B` 。

返回数组 `B` 的最大值和最小值之间可能存在的最小差值。

 

提示：

* \\(1 \le A.length \le 10000\\)
* \\(0 \le A[i] \le 10000\\)
* \\(0 \le K \le 10000\\)
