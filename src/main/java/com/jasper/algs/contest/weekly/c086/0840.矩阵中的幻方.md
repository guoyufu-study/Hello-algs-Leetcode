## 0840. 矩阵中的幻方

[刷题](qu0840/solu/Solution.java)

### 问题描述

\\(3 \times 3\\) 的幻方是一个填充有从 `1` 到 `9` 的不同数字的 \\(3 \times 3\\) 矩阵，其中每行，每列以及两条对角线上的各数之和都相等。

给定一个由整数组成的 `grid`，其中有多少个 \\(3 \times 3\\) 的 “幻方” 子矩阵？（每个子矩阵都是连续的）。

提示:

* \\(1 \le grid.length \le 10\\)
* \\(1 \le grid[0].length \le 10\\)
* \\(0 \le grid[i]\[j] \le 15\\)
